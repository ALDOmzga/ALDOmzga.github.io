FORMATO DE TEXTO

HTML proporciona etiquetas de formato especiales para formatear texto en la página. Podemos usar estas etiquetas para modificar la forma en que se muestra el texto al usuario. Por ejemplo, podemos poner en negrita algunos elementos de texto, o podemos poner en negrita y subrayar elementos importantes, etc. En este tutorial, veamos algunas de las etiquetas de formato más utilizadas en HTML.

Ejemplo 1: texto en negrita Podemos poner texto en negrita usando la etiqueta "b".

<b>Hola mundo<b>

Ejemplo 2: texto en cursiva Podemos poner en cursiva el texto con la etiqueta "i".

<i>Hola mundo<i>

Ejemplo 3: texto en negrita y cursiva Podemos combinar las etiquetas "b" e "i" para que el texto esté en negrita y cursiva.

<b><i>Hola mundo<i><b>

Ejemplo 4: texto en negrita, cursiva y subrayado Podemos subrayar el texto con la etiqueta "u". Aquí probamos las etiquetas "b", "i" y "u".

<u><b><i>Hola mundo<i><b><u>
